This repository contains examples of using Data mining for the following tasks:
1. Classification - to predict the discrete class that the variable of interest falls in
2. Regression - to predict the (continuous) value of the variable of interest
3. Market Basket Analysis - to predict rules that help determine probable antecedents for precedents
4. Sentiment Analysis - to predict overall sentiment on textual data
5. Time Series Prediction - to predict the value of the variable of interest over time
